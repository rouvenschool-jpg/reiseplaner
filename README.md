# Reisegepäck-Planer

Der Reisegepäck-Planer ist eine Web-App mit React und TypeScript.

Mit der App kann der Benutzer seine Reise planen und automatisch eine Packliste generieren.

## Funktionen

- Eingabe von Name und Destination
- Auswahl von Reisedauer
- Auswahl von Klima
- Auswahl von Aktivität
- Auswahl von Transportmittel
- Automatische Generierung einer Packliste
- Checkboxen zum Abhaken
- Validierung: Alle Felder müssen ausgefüllt sein

## Technologien

- React
- TypeScript
- CSS

## Projekt starten

```bash
npm install
npm run dev
